# Horn type and genders of Soay Sheep

Horn type and genders of Soay Sheep *Ovis aires*

## Usage

``` r
data(SShorns)
```

## Format

a data frame with 666 rows and 3 columns, with individual idenitifier
(`id`), horn type (`horn`) and gender (`sex`).

## References

Clutton-Brock T., Pemberton, J. Eds. 2004 Soay Sheep: Dynamics and
Selection in an Island Population
