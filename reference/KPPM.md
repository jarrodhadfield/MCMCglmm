# Kronecker Product Permutation Matrix

Forms an mk x mk Kronecker Product Permutation Matrix

## Usage

``` r
KPPM(m, k)
```

## Arguments

- m:

  integer

- k:

  integer

## Value

Matrix

## References

Schott, J.R.(2003) Journal of Multivariate Analysis 87 (1) 177-190

## Author

Jarrod Hadfield <j.hadfield@ed.ac.uk>
